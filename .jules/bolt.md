## 2025-05-15 - [Vectorized Distance Calculation in BasicEstimator]
**Learning:** Using the squared distance expansion formula (||a-b||^2 = ||a||^2 + ||b||^2 - 2ab) allows vectorizing Euclidean distance calculations across multiple query and fitted embeddings simultaneously. This significantly reduces Python loop overhead and leverages optimized BLAS routines via `np.dot`. Pre-calculating fitted embedding norms during `fit` further improves `predict` performance. However, this formula can produce tiny negative values due to floating-point precision issues, requiring `np.maximum(dists_sq, 0)` for stability.
**Action:** When implementing distance-based estimators, prioritize vectorized matrix operations over iterative `np.linalg.norm` calls, and always handle potential numerical underflow when using the expansion formula.
